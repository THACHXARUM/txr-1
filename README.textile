h1. Transfer Files to and from team members through Web Sockets

h2. Install & Setup

bc. npm install -g txr

h2. Server

bc. txr-server

h2. Client

h3. Send a file to someone listening for files to be sent to them

bc. txr send -u yourFriendsUniqueUsername -f /path/to/your/file

h3. Listen for files

bc. txr listen -u yourUniqueUsername
